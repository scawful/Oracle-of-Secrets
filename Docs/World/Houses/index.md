## Key Houses

### Link's House (0x01)
- **Location:** OW 0x32 (Beach area)
- **Significance:** Starting location after intro
- **Contents:** Bed (save point), storage

### Smith's House (0x64)
- **Location:** OW 0x22 (Smithy area)
- **Significance:** Sword upgrades and tempering
- **Services:** Weapon upgrades

### Village Library (0x49)
- **Location:** OW 0x23 (Village)
- **Significance:** Lore and hints
- **Contents:** Books with game hints

### Chest Minigame (0x67)
- **Location:** OW 0x18 (Forest)
- **Significance:** Gambling minigame
- **Rewards:** Rupees, random prizes
